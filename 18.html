for $=finish.lines -> $line {
    say $line if !($line ~~ m/^\;/ ff $line ~~ m/^\"/);
}

## `$line ~~ m/^\;/ ff $line ~~ m/^\"/` 过滤掉 `;` 和 `"` 之间的内容, 再对它进行取反操作就是过滤后剩下的文本。
